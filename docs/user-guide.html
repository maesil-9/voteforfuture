침착투표소의 세 가지 약속

  • 비밀투표 — 누가 어떤 후보를 골랐는지는 방장(관리자)을 포함해 그 누구도 볼 수 없어요.
  • 1인 1표 — 같은 닉네임으로는 두 번 투표할 수 없어요.
  • 결과 봉인 — 결과 발표 시각이 되기 전에는 누구도 미리 개표할 수 없어요.

이 안내서를 읽는 분

역할하는 일읽을 부분
방장 (관리자)선거를 만들고, 투표를 검수하고, 결과를 공개1 ~ 6단계 전부
투표자 (방 멤버)닉네임 적고 투표「투표하는 방법」, 「현황 보기」

1단계선거 만들기 — 방장용

  1. 관리자 페이지에 접속해요: 사이트주소/admin/login
  2. 전달받은 관리자 이메일과 비밀번호로 로그인해요.
  3. 대시보드에서 「새 선거 만들기」 버튼을 눌러요.
  4. 아래 항목을 채우고 「선거 만들기」를 눌러요.
항목설명
선거명예: [침착한 일상 이야기방] 방장 선거
소개 / 공고문첫 화면에 보일 안내 문구 (선택)
투표 시작 / 종료투표할 수 있는 기간이에요. 이 시간 안에만 투표가 가능해요.
결과 발표이 시각이 되어야 개표 결과가 열려요. 종료 시각보다 뒤여야 해요.
상태아래 표를 참고하세요.
예상 유권자 수방 인원수를 적으면 투표율(%)이 계산돼요. (선택)
상태뜻
draft (작성 중)방장만 볼 수 있어요. 준비하는 동안 이 상태로 두세요.
scheduled (공고됨)멤버들에게 보이지만 아직 투표는 안 돼요. 일정이 되면 자동으로 투표가 열려요.
open / closed보통은 건드릴 필요 없어요. 일정이 알아서 처리해요. (closed로 바꾸면 강제 종료)
archived (보관)모두 끝난 선거를 정리할 때 사용해요.
준비가 끝나면 상태를 draft → scheduled로 바꿔 저장하세요. 시작 시각이 되면 자동으로 투표가 열려요.

2단계후보 등록하기 — 방장용

  1. 선거 관리 화면에서 「후보」 탭을 눌러요.
  2. 「새 후보 등록」에서 이름을 적고 등록해요. 슬로건·짧은 소개·상세 프로필·상징색은 선택이에요.
  3. 등록된 후보 카드에서 공약을 하나씩 추가해요. (제목 + 내용)
  4. 포스터 이미지를 올리면 투표 화면이 훨씬 보기 좋아져요. (3MB 이하)
후보 이름·공약·포스터는 투표가 시작되기 전까지만 고칠 수 있어요. 시작 후에는 잠겨요.

3단계공유 이미지 올리기 — 방장용, 선택

선거 관리의 「개요」 탭 아래쪽 「URL 공유 이미지」에서 이미지를 올려두면, 카카오톡에 링크를 공유할 때 예쁜 미리보기가 함께 보여요. (권장 크기 1200×630)

4단계투표 열고 링크 공유하기 — 방장용

  1. 시작 시각이 되면 투표소가 자동으로 열려요.
  2. 오픈채팅방에 사이트 주소를 공유해요: 사이트주소 (첫 화면에 입장 버튼이 있어요)

투표하는 방법 — 모든 멤버용

  1. 공유받은 링크에 들어가 「닉네임 적고 투표하러 가기」를 눌러요.
  2. 방에서 쓰는 닉네임을 그대로 적고 입장해요.
  3. 후보 포스터를 눌러 소개와 공약을 읽어봐요.
  4. 마음을 정했으면 「이 후보를 선택」→「선택 확인으로」를 눌러요.
  5. 확인 화면에서 한 마디(50자, 선택)를 남길 수 있어요. 다음 방장이나 우리 방에게 하고 싶은 말을 적어보세요.
  6. 「투표 제출하고 봉인하기」를 누르면 끝! 접수 도장이 찍힌 화면이 나와요.
제출한 표는 바꾸거나 취소할 수 없어요. 신중하게 골라주세요.
닉네임은 "내가 투표했다"는 확인에만 쓰여요. 어떤 후보를 골랐는지와는 절대 연결되지 않아요. 한 마디 메시지도 이름 없이 봉인돼요.

5단계검수하기 — 승인과 무효 — 방장용

멤버들이 투표하면 「검수」 탭에 닉네임 목록이 쌓여요. 방장은 이 이름이 진짜 우리 방 멤버인지 확인하고 결정해요. 표의 내용(누굴 찍었는지)은 보이지 않고, 한 마디 메시지도 "있는지 없는지"만 보여요.

버튼언제 누르나요?그러면 어떻게 되나요?
승인 우리 방 멤버가 맞을 때 표가 익명 투표함에 들어가요. 이때부터 그 표는 누구 것인지 아무도 알 수 없어요.
무효 명단에 없는 이름, 오타, 장난 제출일 때 표는 열어보지 않은 채 버려져요. 그 닉네임으로 다시 투표할 수 있게 돼요.
전체 승인 대기 중인 이름이 모두 멤버가 맞을 때 대기 중인 표를 한 번에 모두 승인해요.
승인과 무효는 되돌릴 수 없어요. 승인 전에 이름을 꼭 확인하세요. 그리고 결과 발표 전에 검수를 끝내주세요 — 검수가 안 된 표는 개표에 포함되지 않아요.
누가 내 이름으로 먼저 투표했다는 제보가 오면? 검수 탭에서 그 제출을 무효 처리하세요. 그러면 본인이 다시 투표할 수 있어요.

투표 현황 보기 — 모든 멤버용

아래 화면들에서 실시간 현황(접수 / 승인된 표 / 검수 대기 / 무효 처리)을 누구나 볼 수 있어요. 방장이 승인이나 무효를 처리하면 약 10초 안에 숫자가 바뀌어요.

  • 첫 화면 (사이트주소)
  • 투표를 마친 직후의 완료 화면
  • 결과 페이지 (발표 전에는 봉인 안내와 현황만 보여요)

보이는 것은 숫자뿐이에요. 누가 투표했는지, 누구를 찍었는지는 보이지 않아요.

6단계결과 공개 — 자동

  1. 설정해둔 결과 발표 시각이 되면, 봉인이 자동으로 풀려요. 封 → 개표
  2. 결과 페이지와 관리자 「결과」 탭에서 당선자·득표수·득표율을 볼 수 있어요.
  3. 최다 득표가 같으면 동률로 표시돼요. 이후 처리(결선 등)는 방 규칙으로 정해요.
  4. 관리자 「결과」 탭에는 투표자들이 남긴 익명 한 마디 모음도 함께 보여요. 다음 운영에 참고해보세요.

자주 묻는 질문

방장은 제가 누굴 찍었는지 알 수 있나요?
아니요. 표는 제출 즉시 암호로 봉인되고, 승인되는 순간 이름과의 연결이 완전히 사라져요.
결과를 미리 볼 수 있는 방법이 있나요?
없어요. 발표 시각 전에는 시스템 자체가 개표를 거부해요. 방장도 못 봐요.
닉네임을 잘못 적고 제출했어요.
방장에게 알려주세요. 방장이 그 표를 무효 처리하면 올바른 닉네임으로 다시 투표할 수 있어요.
같은 닉네임으로 두 번 투표하면요?
두 번째 제출은 자동으로 막혀요. "이미 접수되었습니다" 안내가 떠요.
한 마디 메시지는 누가 보나요?
결과 발표 후에 이름 없이 모아서 공개돼요. 검수 중에는 방장도 내용을 볼 수 없어요.
투표 도중에 창을 닫았어요.
제출 전이라면 괜찮아요. 다시 들어가 닉네임을 적으면 처음부터 진행할 수 있어요.

꼭 기억해주세요

  • 제출한 표는 변경·취소 불가 / 승인·무효도 되돌릴 수 없음
  • 후보 정보 수정은 투표 시작 전까지만
  • 검수는 결과 발표 전에 끝내기
  • 표의 내용은 그 누구도 미리 볼 수 없음 — 이것이 침착투표소의 존재 이유예요
